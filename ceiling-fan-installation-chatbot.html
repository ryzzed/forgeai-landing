Frequently Asked Questions

How does a chatbot help ceiling fan installation companies book more jobs?

What wiring is needed to install a ceiling fan with light?
Independent light and fan control requires a 3-wire circuit or a single-wire circuit with a remote control receiver. Older homes often have only a single switched hot wire, requiring either a remote receiver or new wiring. ChatFlow asks about switch configuration to flag homes that need upgraded wiring before the installation quote.
Can a ceiling fan be installed where there is no existing light fixture?
Yes, but it requires new wiring and a properly rated electrical box — increasing the project scope significantly. ChatFlow identifies new-location installs versus fixture replacements so your electrician arrives knowing whether it is a 30-minute swap or a half-day wiring project, enabling accurate upfront quotes.
