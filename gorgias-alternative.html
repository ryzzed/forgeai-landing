Common questions

Can ChatFlow look up order status like Gorgias does?

ChatFlow doesn't have a native Shopify order API integration. It can answer order-related questions based on what you've described in the system prompt (return window, how to initiate a return, shipping timelines) and direct customers to their order confirmation email or tracking link. For live order status lookup ("where is my order #1234?"), Gorgias's Shopify integration is more capable. ChatFlow is better suited for pre-purchase FAQ and post-purchase policy questions.

When should I use Gorgias instead of ChatFlow?

Use Gorgias if you're a high-volume Shopify store processing 100+ support tickets per day and need a team inbox, ticket assignment, macros, and deep order data access. Gorgias is purpose-built for that use case. Use ChatFlow if you're a smaller store without a dedicated support team, want AI to handle common questions automatically, and don't need the full Gorgias helpdesk infrastructure.

How does ChatFlow pricing compare to Gorgias?

Gorgias Starter is $10/month for 50 tickets (very limited). Basic is $60/month for 300 tickets. Pro is $360/month for 2,000 tickets. Plus $0.40 per additional ticket. ChatFlow Starter is $39/month flat for 5,000 AI conversations — no per-conversation fees. For most small stores, ChatFlow's flat pricing is more predictable and covers much higher conversation volume.
