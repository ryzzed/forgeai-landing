WordPress

Use the official ChatFlow plugin — no code needed.

1

Install the ChatFlow plugin

In your WordPress admin, go to Plugins → Add New and search for "ChatFlow". Click Install, then Activate.

Or download directly: github.com/ryzzed/chatflow-wp-plugin/releases/latest

2

Enter your Bot ID

Go to Settings → ChatFlow. Paste your Bot ID (from the Deploy step at myflow.chat) and click Save Settings.

3

Done — your chatbot is live

Visit your site. The chat button appears in the bottom-right corner on every page. No code required.

Any HTML website

Works with any site — static HTML, custom CMS, React, Vue, etc.

1

Copy the snippet

Replace YOUR_BOT_ID with your actual Bot ID from the Deploy step.

<script src="https://chatflow-api-406c.onrender.com/widget.js" data-bot-id="YOUR_BOT_ID" defer></script>
2

Paste it before </body>

Open your HTML template (usually index.html or a layout file) and paste the snippet just before the closing </body> tag. The defer attribute means it loads after your page — zero impact on speed.

3

Deploy and test

Save and publish your site. You should see the chat button appear in the bottom-right corner within a few seconds of the page loading.

Shopify

Add ChatFlow to your Shopify store without editing theme files.

1

Open Theme Code Editor

In your Shopify admin, go to Online Store → Themes. Click the ⋯ menu next to your live theme and select Edit code.

2

Find theme.liquid and paste the snippet

Under Layout, click theme.liquid. Find the closing </body> tag and paste this just before it:

<script src="https://chatflow-api-406c.onrender.com/widget.js" data-bot-id="YOUR_BOT_ID" defer></script>
3

Save and preview

Click Save at the top right. Visit your store — the chat button should appear on all pages including product pages, the cart, and the homepage.

Webflow

Add ChatFlow via Webflow's Custom Code panel — no Designer skills needed.

1

Open Project Settings

In the Webflow Designer, click the W logo (top left) → Project Settings → Custom Code tab.

2

Paste in the "Footer Code" field

Scroll to the Footer Code section and paste:

<script src="https://chatflow-api-406c.onrender.com/widget.js" data-bot-id="YOUR_BOT_ID" defer></script>
3

Save and Publish

Click Save Changes, then Publish your site. The widget appears on all published pages automatically.

Squarespace

Squarespace supports custom code injection on Business and Commerce plans.

1

Open Code Injection

In your Squarespace dashboard, go to Settings → Advanced → Code Injection.

Note: Code Injection requires a Business or Commerce plan.

2

Paste in the "Footer" field

Find the Footer text area and paste the snippet there:

<script src="https://chatflow-api-406c.onrender.com/widget.js" data-bot-id="YOUR_BOT_ID" defer></script>
3

Save and visit your site

Click Save. Open your site in a new tab — the chat button will appear on all pages.

Wix

Add ChatFlow via Wix's Custom Code panel (available on paid plans).

1

Go to Custom Code

In your Wix dashboard, go to Settings → Custom Code (under the Advanced section).

2

Add new code snippet (Body — end)

Click + Add Custom Code. Paste the snippet, set placement to Body — end, and apply to All Pages:

<script src="https://chatflow-api-406c.onrender.com/widget.js" data-bot-id="YOUR_BOT_ID" defer></script>
3

Apply and publish

Click Apply, then Publish your site. The chat button will appear on all pages.
