Switching from LiveChat?

What happens to my existing chat history if I switch?

ChatFlow starts fresh — it doesn't import LiveChat history. But your bot is trained on your business description going forward. Most businesses find the old chat history useful for training content rather than migration.

Does ChatFlow have proactive chat triggers like LiveChat?

Not currently — the widget opens on visitor click. Proactive chat pop-ups are on the roadmap. The bot handles conversations fully once opened, with no agent needed to respond.

Can I run ChatFlow at the same time as LiveChat during evaluation?

Yes. Both are separate script tags. Run them in parallel on different pages or for a test period to compare results before switching.

What platforms does ChatFlow work on?

WordPress, Wix, Squarespace, Webflow, Shopify, or any HTML site that accepts a script tag. No plugin installation required.
